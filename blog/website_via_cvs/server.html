Server Setup

Overview

I am going to make these instructions as general as possible, but for reference the software that I am setting this system up on is:

  • Redhat Linux 7.1 (www.redhat.com) with the latest updates as of 2001/08/07

  • OpenSSH 2.9p1 (www.openssh.org)

  • CVS 1.11 (www.cvshome.org)

  • Apache 1.3.19 (httpd.apache.org)

  • Mailman 2.0.5 (www.list.org)

There have been bugfixes that affect this setup. I know that I had to upgrade to this version of OpenSSH to allow a users ssh configuration files to be owned by root rather than the user. In general it is a good idea to upgrade to the latest versions of programs especially since much of the time things that are fixed are security vulnerabilities.
