This pages covers the work generated by Andy Trent, Danyel Bruggink, and Will Holcomb in the course of their work in Computer Science 370, software design, spring semester 1999 at Tennessee Technological University.

In addition to a project webpage there was also several pieces of documentation developed specifically for the class. There were:

  • The proposal document
  • The sotware design specification
  • The final design document

In addition I developed a set of javadocs which document the classes that I developed to parse the input commands.

Also there is the work on the original java robot. Additionally, the actual project work is viewable online if you have the java 3D extension api installed for your browser.
