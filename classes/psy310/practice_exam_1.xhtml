  13. Suppose you have evenly spaced numbers, and the distance between adjacent numbers is k. It is well known that, if N = 3, the sample mean of the 3 numbers is the middle value and their sample variance is k2. Suppose N = 5. Find an expression for the sample variance of the 5 numbers.

    4. 2.5k2 ✓

    First, consider the sum of a sequence of numbers:

    ∑ i=1 k i = 1+2+…+ k-1+ k = 1+k+ 2+k-1 +…+ k-12+ k- k-12-1 +k+12 = k-12 k+1+ k+12 = k2+k 2

    The same basic logic produces:

    ∑ i=1 k i2 = k3+ 2k2 2

    This can be used in the expansion of n elements evenly spaced by k units:

    ∑ i=1 n xi = x1+ x1+k+ x1+2k +…+ x1+ n-1k = nx1+ k+2k+…+ n-1k = nx1+k ∑ i=1 n-1 i = nx1+ k n-12+ n-1 2 = nx1+k n2-n 2

    And again, through a similar process:

    ∑ i=1 n xi2 = nx12+ kx1 ∑ i=1 n-1 i+ k2 ∑ i=1 n-1 i2 = nx12+ kx1 n2-n 2 +k2 n3-n2+n 2

    Consider then the mean:

    μ = ∑ i=1 n xi n = x1+k n-1 2

    The variance then is:

    σ2 = ∑ xi-μ 2 n-1 = ∑ xi2- μxi+ μ2 n-1 = ∑xi2- ∑μxi+ ∑μ2 n-1 = ∑xi2- μ∑xi+ nμ2 n-1
