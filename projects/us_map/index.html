Clickable US Map

This is a set of scripts to generate a clickable image map from the following image:

If you want to look at it more closely, use Batik via webstart. The scripts used in generating are:

  • hide_elements.js — A javascript for the SVG which hides everything except a selected state
  • trim_image.py — The images generated from the hidden elements are mostly transparent. This uses the python image library to crop the transparent edges out
  • process_map.py — Generates a convex hull for each image using Graham's scan to produce the HTML
