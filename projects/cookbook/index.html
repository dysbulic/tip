PC-RIM Cookbook

Long ago the Peace Corps volunteers of Mauritania put together a cookbook. Unfortunately the electronic version of this document was lost to the annals of time. I scanned in all the pages and set about reconstructing it. A couple of volunteers at the time said that they were going to do the drudgery of collecting and typing in all the recipes. So, my task was to make it quick and easy to disseminate that information. Specifically, what were were looking for was:

  • Printable Format: Something pretty to print and keep in regional houses. This document should have:
    • Sections: Groupings of recipes by things such as Meats, Vegetables, Desserts, whatever
    • Table of Contents: Every recipe listed up front with the appropriate page numbers
    • Indexes: Alphabetical by name of dish and also groupings by main ingredients
  • Online Format: Something if the print copy gets lost of if RPCVs want some hakko back in the States. This had the same basic requirements as the print version, but would ideally be in HTML

The issue was I have little tolerance for maintaining things like indexes by hand. Particularly if I add a meat recipe halfway through the book and it shifts the following 25 recipes over a page. Not only is it a pain in the ass, it's really error prone.

So, I started on a project to take the recipes, collect them in some format and let the computer deal with making the documents. I made some progress before the project floundered for lack of time:

  • cookbook.docbook — This is the overarching document structure. Where to put the table of contents, how the sections are ordered, etc.
  • recipes/ — A collection of recipes in Recipe Markup Language. These are only new recipes not in the original cookbook because theoretically the other volunteers were reentering the old recipes.
  • cookbook.xslt — XSLT program to take cookbook.docbook and generate a XSL:FO file. This can then be used to generate printable PDF cookbook.
  • rml-fo.xslt — XSLT program to generate individual printable recipes from the RML recipes.
  • rml-xhtml.xslt — XSLT program to generate individual HTML recipes from the RML recipes.
