Firefox Greasemonkey Scripts

There are two user scripts available for Greasemonkey which allow you to remove the sIFR elements from a page. This can be useful if, for example, you are using FlashBlock, which blocks these elements.

The first script provides some tools such as removing the sIFR elements and a way to execute the replacement statements if you're debugging. It requires GreaseMonkey 0.2.6. sIFR Tools.
