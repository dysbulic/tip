Installing Podcast Joomla Component

  1. Go to http://ebene.madstones.com/indiefeed/
  2. Log In
  3. Installers → Components
  4. chown www-data:www-data -R /var/www/joomla/
  5. Package: /home/will/sites/himinbi.org/odin/podcast_component
  6. The alphabetically first xml file will be used as the installation file
  7. Menu → mainmenu
  8. New → Component → Podcast → Save
  9. Reorder to top
  10. ln -s ../components/com_podcast/simplicity/ from joomla/templates/
  11. Unpublish modules in Module Manager
  12. mv /var/www/joomla/htaccess.txt /var/www/joomla/.htaccess
  13. Global Configuration → SEO → Search Engine Friendly URLs
