Set is a deceptively simple game played with a deck of 81 cards where each card differs from all the others on at least one of four axes:

  • Color: Red, Green or Purple
  • Number: One, Two or Three
  • Fill: Hollow, Lines or Solid
  • Shape: Oval, Squiggle, Diamond

A valid set consists of three cards which are either all the same on a given axis or all different. Since no card is repeated in the deck, a set must have at least one axis which differs. For example, a simple set is alike in color, shape and shade, but different in number:
