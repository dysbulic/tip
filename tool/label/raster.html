These are some cd labels I worked up while waiting on my debian discs to download. These are kinda unique becasue instead of using tux they use the Linux fox by Ed Mackey rather than tux. I was looking for something a bit cooler looking (no offense to tux). =)

  • Disc 1
  • Disc 2
  • Disc 3

These were designed to go on Avery 5824 CD Labels. If you use the Avery printing software (under Windows) to print them then debian-label.dsn is the configuration file for that. This image was composited in Photoshop and debian-cover-expanded.psd is the original.

I collected several images from several places while contemplating this. Those along with the fonts are in ./images/
